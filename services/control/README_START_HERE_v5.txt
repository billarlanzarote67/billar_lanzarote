BILLAR LANZAROTE – MASTER CONTROL PACK v5

Purpose
- One-click install
- One-click start/stop/status
- Strict fail-fast validation
- RTSP route checks
- Live dashboard
- Log cleanup and readable debugging

Recommended use
1. Install this pack into C:\AI\BillarLanzarote
2. Run 00_INSTALL_MASTER_CONTROL_PACK_v5.bat
3. Run INSTALL_ALL_v5.bat
4. Run START_ALL_v5.bat
5. Open dashboard
6. If something hangs:
   - CHECK_SYSTEM_HEALTH_v5.bat
   - VIEW_LOGS_v5.bat
